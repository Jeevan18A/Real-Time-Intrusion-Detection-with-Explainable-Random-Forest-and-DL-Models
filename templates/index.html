About This System

This Intrusion Detection System uses advanced machine learning techniques to analyze network traffic in real-time. It employs:

  • Packet Sniffing: Captures network packets using Scapy
  • Flow Analysis: Groups packets into flows for comprehensive analysis
  • ML Classification: Random Forest model for attack detection
  • Anomaly Detection: Autoencoder for identifying unusual patterns
  • Explainability: LIME for model interpretation
